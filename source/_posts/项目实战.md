启动类

```
@RestController
@SpringBootApplication
public class DoaApplication {

	@Resource
	private JdbcTemplate jdbcTemplate;

	public static void main(String[] args) {
		SpringApplication.run(DoaApplication.class, args);
	}

	@GetMapping("/test")
	public List<Map<String, Object>> test() {
		return jdbcTemplate.queryForList("select * from user");
	}
}

```

配置

```
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/doa?useUnicode=true&characterEncoding=utf-8&useSSL=false&rewriteBatchedStatements=true&serverTimezone=UTC
    username: root
    password: root
```



```
<-- pom -->
<dependency>
			<groupId>mysql</groupId>
			<artifactId>mysql-connector-java</artifactId>
		</dependency>
```
